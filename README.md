# 光伏电站远程运维平台 (solar-ops-platform)

## 项目简介

光伏电站远程运维平台是一个集**设备数据采集**、**实时监控**、**发电效率分析**、**故障工单自动生成**于一体的智能化运维管理平台。平台采用微服务架构，支持多端（PC管理端 + 移动端）协同作业，解决光伏电站运维响应慢、效率评估滞后的问题。

## 技术架构

### 后端技术栈
- **框架**: Spring Boot 2.7.18 + Java 8
- **ORM**: MyBatis-Plus 3.5.3
- **数据库**: MySQL 8.0（业务数据） + InfluxDB 2.x（时序数据）
- **缓存**: Redis + Redisson
- **消息队列**: RocketMQ
- **设备接入**: Netty（TCP） + EMQX（MQTT）
- **接口文档**: Knife4j
- **认证**: JWT

### 前端技术栈（PC管理端）
- **框架**: React 18 + Vite
- **UI组件库**: Ant Design 5
- **图表库**: ECharts 5
- **路由**: React Router 6
- **HTTP请求**: Axios
- **实时通信**: WebSocket

### 移动端技术栈
- **框架**: uni-app + Vue 3
- **UI组件**: uView UI 风格
- **构建工具**: Vite
- **支持平台**: H5 / 微信小程序 / App

## 快速开始

### 环境要求
- JDK 1.8+
- Maven 3.6+
- Node.js 16+
- MySQL 8.0+
- Redis 6.0+
- InfluxDB 2.x
- RocketMQ 5.x
- EMQX 5.x（可选）

### 数据库初始化

```bash
# 执行初始化脚本
mysql -u root -p < docs/sql/init.sql
```

### 后端启动

```bash
# 编译整个项目
mvn clean install -DskipTests

# 启动管理后台服务
cd solar-ops-admin
mvn spring-boot:run

# 启动设备接入服务
cd ../solar-ops-device
mvn spring-boot:run

# 启动效率分析服务
cd ../solar-ops-analysis
mvn spring-boot:run

# 启动工单管理服务
cd ../solar-ops-workorder
mvn spring-boot:run
```

### PC管理端启动

```bash
cd solar-ops-web
npm install
npm run dev
```

访问地址: http://localhost:5173

默认账号: admin / 123456

### 移动端启动

```bash
cd solar-ops-app
npm install
# H5模式
npm run dev:h5
# 微信小程序
npm run dev:mp-weixin
```

## 关键技术点

### 数据延迟保障
- 设备数据 → InfluxDB：< 1秒
- InfluxDB → WebSocket 推送：< 2秒
- 端到端延迟：< 5秒

### PR计算公式
```
PR = 实际发电量 / (装机容量 × 峰值日照小时数)
```

### 健康度分级
| 等级 | PR值范围 | 标识颜色 |
|------|---------|---------|
| 优秀 | PR ≥ 0.85 | 🟢 绿色 |
| 良好 | 0.75 ≤ PR < 0.85 | 🟡 黄色 |
| 较差 | PR < 0.75 | 🔴 红色 |

### 工单状态流转
```
待接单(0) → 已接单(1) → 处理中(2) → 待验收(3) → 已完成(4)
                                          ↘ 已关闭(5)
```

### 故障级别
| 级别 | 代码 | 响应时限 |
|------|------|---------|
| 低级 | 1 | 48小时 |
| 中级 | 2 | 24小时 |
| 高级 | 3 | 8小时 |
| 紧急 | 4 | 2小时 |

## 开发规范

### 后端开发规范
- 遵循阿里巴巴Java开发手册
- RESTful API 设计规范
- 统一响应格式：`Result<T>`
- 统一异常处理：`GlobalExceptionHandler`
- 分页统一：`PageQuery` / `PageResult`

### 前端开发规范
- 组件化开发
- 统一请求封装
- 状态管理：React Context / Pinia
- 代码风格：ESLint + Prettier
